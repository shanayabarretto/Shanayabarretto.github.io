Projects

This is a collection of engineering related projects I have worked on in the past or am currently working on. I have learned so much from these projects, and I am really proud of how they turned out.

CPR Guidance Tool

The idea for this project was given to me by one of my teachers during the pandemic. It is a circuit board with an attached pressure sensor that tells the user via LEDs whether their CPR compressions are of the right speed and pressure. I designed the board using eagle, printed it, drilled it, soldered it, and programmed it using Assembly Language.

G2 Gear Up!

My final project for Harvard's CS50x course was G2 Gear Up! which I wrote using Lua. When learning to drive, I had trouble remembering safety checks (mirror, signal, shoulder, etc.) so I made this game to help me. It places you in a common on road scenario and allows you to practice maneuvering through the situation safely. It helped me obtain my G2!

Buddy Backdrops

My friends (Niveditha Renganathan, Kate Bendall, and Cassandra Hopkins) and I decided to create a Chrome Extension recently. We were inspired by the app Locket and wanted to create an extension that allows friends to share a ‘new tab’ chrome background. For this project, I have been working on the backend side, using flask and SQL to help put it together.

Obstacle Course Programming

As might be inferred by the engineering program I chose, I really like working with robots. In high school, my team was the first ever to complete this project. We used C to have the robot follow lines, pickup objects, and read barcodes. We completed a similar challenge last semester during Tron Design Days.

Agent Morse

For an open ended project last semester, my team and I created a secret mission scenario where our robot, Agent Morse, would help spies escape a dangerous situation. Using C++ we programmed the robot such that when fed a message, it could tap it in morse code to a trapped comrade on the other side of a wall.

Robot Design

In my tech design class, we picked a FIRST Lego League competition prompt and designed the optimal robot for it, allowing me to become comfortable with 3d modelling software. I focused on the design of the wheels (which allow the robot to move both horizontally and vertically easily) and overall assembly.

Tranquil Nights

For EngHack last year, my team and I created an app with pygame called Tranquil Nights which functions as a sleep tracker. I also got my first taste of web design during the hackathon, as we made a website to accompany the app. Participating in this hackathon was so much fun, and I intend to attend many more in the future.

Personal Website

I wanted to challenge myself to create a more professional looking website, and build up my web development skills, which is why I decided to create this website. I used bootstrap to help with the styling, and am using flask in this project as well. I am committed to continuously improving this website as I learn more about web development.

Minigame Mine

I also consider coding to be one of my hobbies, and often experiment with code just to learn and have fun. Recently, I have been using languages and tools I don’t know or have not used in a while to create minigames. I have used pygame to make tic tac toe, and java & swing to make hangman so far, and I plan to keep extending this list.
